# KinoBAZA.RU — Онлайн кинотеатр

Современный онлайн-кинотеатр с фильмами и сериалами в стиле IVI.ru.

## Особенности

- 🎬 **Просмотр фильмов и сериалов** через API Кинопоиска
- 🔐 **Система подписки**: 7 дней посещения = 20 дней полного доступа
- 📺 **3 источника видео**: Kinobox, Kinoplayertop, Kinobd.net
- 🎨 **Дизайн в стиле IVI.ru** с градиентами и современной анимацией
- 📱 **Адаптивный дизайн** для мобильных устройств
- 🔍 **Поиск фильмов** по названию
- 📋 **FAQ, Политика конфиденциальности, Документация API**

- **HTML5/CSS3** - современный дизайн с CSS Variables
- **JavaScript (Vanilla)** - без фреймворков
- **KinopoiskAPI** - получение данных о фильмах
- **Kinobox/Kinoplayertop/Kinobd** - плееры для просмотра
- **localStorage** - хранение подписки и настроек

## Как это работает

### Система подписки

1. При первом посещении пользователь не имеет доступа к просмотру
2. Каждый день посещения засчитывается как 1 день активности
3. После 7 дней активности автоматически активируется подписка на 20 дней
4. С активной подпиской доступен просмотр всех фильмов и сериалов

### Плееры

- **Kinobox** (Основной) - kinoboxtv.tatnet.app/kinobox.js
- **Kinoplayertop** (Запасной) - kinoplayertop.tatnet.app/kinoplayer.js  
- **Kinobd** (Резервный) - kinobd.net/js/player_.js

## API

Используется неофициальный API Кинопоиска:

```
Base URL: https://kinopoiskapiunofficial.tech/api
Token: 960dc4e3-9eae-4072-8f0c-98a5ed6c0d2a
```

### Эндпоинты

- `GET /v2.2/films/top?type=TOP_100_POPULAR_FILMS&page=1` - популярные фильмы
- `GET /v2.2/films/top?type=TOP_250_BEST_FILMS&page=2` - лучшие фильмы
- `GET /v2.1/films/search-by-keyword?keyword={query}` - поиск
- `GET /v2.2/films/{id}` - детали фильма

## Установка и запуск

### Локально

1. Откройте `index.html` в браузере
2. Или используйте локальный сервер:
```bash
python -m http.server 8000
# Откройте http://localhost:8000
```

### Деплой

Проект готов к деплою на любой статический хостинг:

- **GitHub Pages**: Запушите репозиторий и включите Pages
- **Vercel/Netlify**: Просто подключите репозиторий
- **Любой веб-сервер**: Загрузите index.html
