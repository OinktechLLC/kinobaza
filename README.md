# KinoBAZA.RU - Онлайн кинотеатр

Современный онлайн-кинотеатр в стиле ivi.ru с обязательной системой подписки.

## 🎬 Особенности

- **Дизайн**: Тёмная тема в стиле ivi.ru с фиолетовыми градиентами
- **Подписка**: Бесплатная на 20 дней с условием посещения каждые 7 дней
- **Плееры**: Kinobox, KinoPlayerTop, KinoBD.net
- **API**: Интеграция с Кинопоиском для трейлеров
- **Cookies**: Простая система (kinobaza_consent, kinobaza_sub, kinobaza_exp, kinobaza_visit)

## 🔐 Система подписки

### Как работает:
1. Подписка активируется на **20 дней**
2. Необходимо заходить на сайт **каждые 7 дней** для продления активности
3. Если не зайти в течение 7 дней - подписка аннулируется
4. При каждом визите дата следующего обязательного посещения обновляется

### Cookies:
- `kinobaza_consent` - согласие на использование cookies
- `kinobaza_sub` - статус подписки (active/not active)
- `kinobaza_exp` - дата истечения 20-дневного периода
- `kinobaza_visit` - дата следующего обязательного визита (7 дней)

## 📁 Структура

Все реализовано в **одном HTML файле** (`index.html`):
- Главная страница с каталогом фильмов
- Модальное окно плеера с 3 источниками
- Модальное окно оформления подписки
- Страницы: FAQ, Политика конфиденциальности, Docs
- Cookie-баннер
- Адаптивный дизайн для мобильных

## 🚀 Запуск

Просто откройте `index.html` в любом браузере. Сервер не требуется.

## 📱 Страницы

- **Главная** - каталог фильмов и сериалов
- **FAQ** - ответы на вопросы о подписке
- **Политика** - политика конфиденциальности
- **Docs** - документация по интеграции плееров

## ⚠️ Важно

Для просмотра любого контента необходима активная подписка. 
Без подписки открывается только модальное окно с предложением оформить её.
