# 🎬 KinoBAZA.RU — Онлайн кинотеатр

Современный онлайн-кинотеатр в стиле ivi.ru с интеграцией API Кинопоиска.

## ✨ Особенности

- **Дизайн в стиле ivi.ru** — фиолетово-синяя цветовая гамма, современные карточки, анимации
- **API Кинопоиска** — актуальная информация о фильмах и сериалах
- **Система подписки** — 20 дней доступа = 7 дней активного использования
- **Три видеоплеера** — Kinobox, KinoBD.net, KinoPlayerTop
- **Трейлеры** — интеграция с YouTube через API
- **Адаптивность** — работает на всех устройствах
- **Один HTML файл** — всё включено, никаких зависимостей

## 🚀 Быстрый старт

1. Откройте `index.html` в любом современном браузере
2. Примите cookies
3. Активируйте подписку (бесплатно)
4. Наслаждайтесь просмотром!

## 🔐 Система подписки

Подписка обязательна для просмотра контента:

```
20 дней подписки = 7 дней активного посещения сайта
```

### Cookies
| Название | Описание | Срок |
|----------|----------|------|
| `kinobaza_consent` | Согласие на cookies | 365 дней |
| `kinobaza_sub` | Статус подписки | 20 дней |
| `kinobaza_exp` | Дата истечения | 20 дней |
| `kinobaza_visit` | Счётчик визитов | 20 дней |

## 🛠 Технические детали

### API Configuration
```javascript
const API_KEY = '960dc4e3-9eae-4072-8f0c-98a5ed6c0d2a';
const API_BASE = 'https://kinopoiskapiunofficial.tech/api/v2.2';
```

### Используемые эндпоинты
- `/films/top?type=TOP_POPULAR_ALL` — популярные фильмы
- `/films/top?type=TOP_POPULAR_SERIES` — популярные сериалы
- `/films/searchByKeyword?keyword={query}` — поиск
- `/films/{id}/videos` — трейлеры фильма

### Источники видео
```javascript
// Kinobox (основной)
https://kinobox.tv/embed/{kinopoiskId}

// KinoBD.net (резервный)
https://kinobd.net/embed/{kinopoiskId}

// KinoPlayerTop (запасной)
https://kinoplayer.top/embed/{kinopoiskId}
```

## 🎯 Как это работает

1. При загрузке проверяются cookies подписки
2. Если подписки нет — показывается модальное окно
3. После активации — загрузка фильмов через API Кинопоиска
4. При клике на фильм — проверка подписки → открытие плеера
5. Переключение между источниками видео без перезагрузки
