Политика конфиденциальности

1. Сбор данных

KinoBAZA.RU использует cookies и localStorage только для сохранения предпочтений. Мы не собираем персональные данные.

2. IP-адрес

Ваш IP используется только для определения региона и не сохраняется.

3. Сторонние сервисы

При просмотре IP может передаваться плеерам Kinobox, Kinoplayertop, Kinobd.net.

4. API Кинопоиска

Мы используем неофициальный API Кинопоиска для получения информации о фильмах.
